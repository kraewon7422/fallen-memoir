# FALLEN MEMOIR

> 한국어 픽셀아트 던전 탈출 게임. 단일 HTML + Canvas 2D + JavaScript로 만들어졌으며, 외부 이미지 에셋 없이 모든 그래픽을 코드로 그립니다. PC·모바일 브라우저에서 바로 플레이할 수 있습니다.

🎮 **바로 플레이 → `https://kraewon7422.github.io/fallen-memoir/`**

---

## 게임 소개

오래된 동굴 옆 마을에 사는 주인공이, 깊은 미로 속으로 내려가며 그 끝에 잠든 진실을 마주하는 이야기입니다. 던전을 한 층씩 탐험하고, 5층마다 등장하는 보스를 쓰러뜨리며 더 깊이 내려갑니다. 미로 곳곳의 책장에 남겨진 고서를 읽으면 이 세계가 어쩌다 이렇게 되었는지 조금씩 드러납니다.

두 개의 던전(돌로 된 제1던전 / 사막 던전)이 있고, 각각 30층의 최종 보스를 향해 나아갑니다.

## 주요 특징

- **절차적 던전 생성** — 들어갈 때마다 새로 만들어지는 미로
- **층마다 다른 도전** — 5층마다 보스, 30층 최종 보스, 30층 이후 반복 보스
- **두 갈래 모험** — 돌 던전과 사막 던전, 각기 다른 보스 도감과 이야기
- **전투 시스템** — 근접 무기 + 불·얼음·번개 원소 마법, 무기 강화(최대 +5)
- **아이템** — 다양한 포션(투명·신속·강화·보호막 등)과 상점
- **분위기 연출** — 횃불 조명·시야(fog of war), 8비트 효과음, 화면 흔들림
- **마을 ↔ 던전 ↔ 집** 오가는 씬 구성, 마을 포털과 층 선택 워프 포털
- **온라인 리더보드** — 친구들과 코인·최고 층·처치 수 순위를 실시간 공유
- **자동 저장** — 진행 상황이 브라우저에 저장됨

## 조작법

### PC

| 동작 | 키 |
|---|---|
| 이동 | `WASD` 또는 화살표 |
| 공격 | `SPACE` |
| 인벤토리 / 상자 열기 / 상호작용 | `E` |
| 마을로 이동 | `Ctrl` + `Shift` + `H` |
| 메뉴(처음 화면)로 | `Ctrl` + `Shift` + `E` |

### 모바일

| 동작 | 조작 |
|---|---|
| 이동 | 오른쪽 아래 **조이스틱** 드래그 |
| 공격 | 왼쪽 아래 **⚔ 버튼** (꾹 누르면 연사, 던전에서만 표시) |
| 인벤토리 | **🎒 버튼** (무기 칸 탭 → 한 번 더 탭하면 장착) |
| 상호작용 | 화면 **더블 탭** (상자·NPC·집/동굴 입장·침대) |
| 상점 | 항목 탭으로 선택, 한 번 더 탭하면 구매 |
| 처음 화면으로 | 마을/집에서 왼쪽 위 **≡** 버튼 |

> 세로·가로 모드 모두 지원합니다. 가로 모드에서는 조이스틱과 버튼이 화면 모서리에 반투명으로 겹쳐 표시됩니다.

## 리더보드

시작 화면에서 친구들과 세 가지 순위를 겨룹니다.

- 💰 **코인** — 한 판에서 모은 누적 금화
- ⬇ **내려간 층** — 도달한 최고 깊이
- ⚔ **처치 수** — 쓰러뜨린 적의 수

각 브라우저마다 고유 ID가 부여되어 개인 최고 기록이 등록되며, 닉네임을 바꿔도 기존 기록이 따라옵니다. 사망·탈출·마을 도착(체크포인트)·메뉴 종료 시점에 기록이 제출됩니다.

## 기술

- HTML5 / Canvas 2D / 순수 JavaScript (단일 파일)
- WebAudio API (8비트 효과음)
- Galmuri 픽셀 폰트 (`galmuri-fonts.css`)
- Firebase Realtime Database (리더보드)
